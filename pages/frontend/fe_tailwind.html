install Tailwind CSS in a React project

To install Tailwind CSS in a React project using CLI tools, follow these steps:

1. Create a React Application

If you donâ€™t already have a React app, create one with create-react-app:

npx create-react-app my-app
cd my-app

2. Install Tailwind CSS

Install Tailwind CSS via npm:

npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init

This creates a tailwind.config.js file for configuring Tailwind CSS.

3. Configure Tailwind

Modify the tailwind.config.js file to specify the paths to your template files. Add all locations where your React components will live:

module.exports = {
  content: [
    "./src/**/*.{js,jsx,ts,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
};

4. Add Tailwind to Your CSS

Open the src/index.css file and add the following @tailwind directives:

@tailwind base;
@tailwind components;
@tailwind utilities;

5. Start Your Application

Run the React application to ensure Tailwind CSS is working:

npm start

6. Test Tailwind

Add some Tailwind classes to App.js or any component to verify installation. For example, replace the contents of App.js with:

function App() {
  return (
    <div className="min-h-screen bg-blue-100 flex items-center justify-center">
      <h1 className="text-4xl font-bold text-blue-600">Hello, Tailwind CSS!</h1>
    </div>
  );
}

export default App;

What is a Tailwind CSS class?
A Tailwind CSS class is a utility-first CSS framework class used to style elements directly in your HTML. Each class corresponds to a single style property, making it easy to build complex designs by combining multiple classes without writing custom CSS.
Example of Tailwind CSS Classes

Below is an example of Tailwind CSS classes applied to an HTML element:


                    <div class="p-4 bg-blue-500 text-white rounded-lg shadow-md">
                    Hello, this is a styled div!
                </div>

Here:

  • p-4: Adds padding of 1rem (16px).
  • bg-blue-500: Sets the background color to blue with a specific intensity.
  • text-white: Makes the text color white.
  • rounded-lg: Applies large border-radius to round corners.
  • shadow-md: Adds a medium-sized shadow effect.
