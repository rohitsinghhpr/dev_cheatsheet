CSS Syntax

CSS Syntax refers to the set of rules that define how styles are applied to HTML elements. It consists of selectors, properties, and values.

Basic Syntax:

        selector {
            property: value;
        }

Example:

        h1 {
            color: blue; /* Selects all h1 elements and sets their text color to blue */
        }

Explanation: The selector is the HTML element you want to style, property defines what aspect of the element you want to modify (e.g., color, font-size), and value specifies the actual style to apply.

CSS Selectors

CSS Selectors are patterns used to select and style HTML elements. They target elements based on their type, attributes, or relationships in the DOM.

Types of Selectors:

  • Universal Selector: * selects all elements.
    •         * {
            color: red;
        }
  • Type Selector: Selects elements by their tag name.
    •         h1 {
            font-size: 24px;
        }
  • Class Selector: Selects elements with a specific class attribute.
    •         .my-class {
            color: green;
        }
  • ID Selector: Selects an element with a specific ID.
    •         #my-id {
            margin: 10px;
        }
  • Attribute Selector: Selects elements based on an attribute or its value.
    •         input[type="text"] {
            border: 1px solid black;
        }
  • Descendant Selector: Selects elements that are inside a specific parent element.
    •         div p {
            color: blue;
        }
CSS Pseudo-Classes

CSS Pseudo-Classes are used to define the special state of an element, like when it’s hovered over, focused, or visited. They are prefixed with a colon (:) and target dynamic states.

Examples of Pseudo-Classes:

  • :hover — Applied when an element is hovered over by the user.
    •         a:hover {
            color: red;
        }
  • :focus — Applied when an element gains focus (like an input field).
    •         input:focus {
            border: 2px solid blue;
        }
  • :nth-child() — Targets an element based on its position in a parent element.
    •         li:nth-child(2) {
            color: green;
        }
  • :first-child — Targets the first child element of a parent.
    •         ul li:first-child {
            font-weight: bold;
        }
  • :last-child — Targets the last child element of a parent.
    •         ul li:last-child {
            font-size: 20px;
        }
CSS Pseudo-Elements

CSS Pseudo-Elements are used to style specific parts of an element, such as the first letter or first line of text, or to insert content before or after an element. They are prefixed with two colons (::) in modern browsers.

Examples of Pseudo-Elements:

  • ::before — Inserts content before an element's content.
    •         p::before {
            content: "Note: ";
            font-weight: bold;
        }
  • ::after — Inserts content after an element's content.
    •         p::after {
            content: " (Read more)";
            color: blue;
        }
  • ::first-letter — Styles the first letter of a block-level element's content.
    •         p::first-letter {
            font-size: 2em;
            color: red;
        }
  • ::first-line — Styles the first line of text in a block-level element.
    •         p::first-line {
            font-weight: bold;
        }
CSS Units

CSS Units define the measurement of properties like width, height, margin, padding, etc. Units can be divided into two main types:

  • Absolute Units: These units are fixed and do not change according to the screen size or the user's settings.
    • px — Pixels, the most common unit for screen design.
      •         div {
            width: 200px;
        }
    • cm — Centimeters, used for print-based designs.
    • mm — Millimeters, also used in print-based designs.
    • in — Inches, another print-related unit.
    • pt — Points, used in typography, 1pt = 1/72 of an inch.
    • pc — Picas, used in print, 1pc = 12pt.
  • Relative Units: These units are based on other elements, such as the size of the parent element or the viewport.
    • % — Percentage, relative to the parent element's size.
      •         div {
            width: 50%;
        }
    • em — Relative to the font-size of the element or parent element.
      •         p {
            font-size: 2em; /* 2 times the size of the parent font */
        }
    • rem — Relative to the font-size of the root (html) element.
      •         html {
            font-size: 16px;
        }
        p {
            font-size: 2rem; /* 32px */
        }
    • vw — Viewport width, relative to 1% of the viewport's width.
    • vh — Viewport height, relative to 1% of the viewport's height.
    • vmin — The smaller of the viewport's width or height.
    • vmax — The larger of the viewport's width or height.
CSS Box Model

CSS Box Model describes the rectangular boxes generated for elements, consisting of content, padding, border, and margin areas.

The components of the Box Model are:

  • Content: The actual content of the element, such as text or images.
  • Padding: The space between the content and the border. It is transparent.
  • Border: The boundary surrounding the padding (if any), which can be styled.
  • Margin: The space outside the border, separating the element from other elements. It is also transparent.

Box Model Calculation: The total width of an element is the sum of its content, padding, border, and margin.

        element {
            width: 100px;
            padding: 10px;
            border: 2px solid black;
            margin: 5px;
        }

box-sizing: The box-sizing property defines whether the width and height properties include the padding and border.

        element {
            box-sizing: border-box; /* Includes padding and border in the element's total width and height */
        }
CSS Positions

CSS Positioning determines how an element is positioned within the document. There are several positioning schemes:

  • static: Default value. Elements are positioned according to the normal flow of the document.
    •         div {
            position: static;
        }
  • relative: The element is positioned relative to its normal position, using the top, right, bottom, or left properties. 
    •         div {
            position: relative;
            top: 10px;
            left: 20px;
        }
  • absolute: The element is positioned relative to its nearest positioned ancestor (instead of the viewport), or the body element if there is no positioned ancestor.
    •         div {
            position: absolute;
            top: 50px;
            left: 100px;
        }
  • fixed: The element is positioned relative to the viewport, and remains in place even when the page is scrolled.
    •         div {
            position: fixed;
            top: 10px;
            right: 10px;
        }
  • sticky: The element is positioned relative to the viewport until a specified scroll position is reached, at which point it "sticks" to that position.
    •         div {
            position: sticky;
            top: 0;
        }
CSS Media Queries

CSS Media Queries are used to apply different styles based on the characteristics of the device or viewport, such as its width, height, or resolution. They enable responsive design.

Example: Changing styles based on the screen width:

        @media (max-width: 600px) {
            body {
                background-color: lightblue;
            }
        }

Common Media Query Breakpoints:

  • @media (max-width: 768px) — Target tablets and smaller devices.
  • @media (max-width: 480px) — Target mobile phones.
  • @media (min-width: 1024px) — Target larger screens like desktops.

Example of multiple media queries:

        @media (max-width: 768px) {
            .container {
                width: 100%;
            }
        }
        @media (min-width: 769px) {
            .container {
                width: 80%;
            }
        }
CSS Flexbox

CSS Flexbox is a layout model that allows you to design complex layouts with ease, distributing space within a container and aligning items in various directions.

Flexbox container: The element with display: flex defines a flex container.

        .container {
            display: flex;
        }

Flex properties for the container:

  • flex-direction: Defines the direction of the flex items (row, column, etc.).
    •         .container {
            flex-direction: row; /* default: items arranged horizontally */
        }
  • justify-content: Aligns items horizontally along the main axis (left to right or top to bottom).
    •         .container {
            justify-content: center; /* Center items horizontally */
        }
  • align-items: Aligns items vertically along the cross axis (perpendicular to the main axis).
    •         .container {
            align-items: flex-start; /* Align items to the top */
        }
  • flex-wrap: Allows items to wrap onto the next line if necessary.
    •         .container {
            flex-wrap: wrap; /* Allow items to wrap */
        }

Flex properties for the items:

  • flex-grow: Defines how much a flex item should grow relative to the rest.
    •         .item {
            flex-grow: 1; /* This item will grow to fill available space */
        }
  • flex-shrink: Defines how much a flex item should shrink relative to the rest.
    •         .item {
            flex-shrink: 1; /* This item will shrink if there is not enough space */
        }
  • flex-basis: Defines the initial size of a flex item before growing or shrinking. 
    •         .item {
            flex-basis: 200px; /* The item will start with a width of 200px */
        }
CSS Grid

CSS Grid is a two-dimensional layout system for the web that allows you to create complex layouts using rows and columns.

Grid Container: To create a grid layout, you set the display property to grid. 

        .container {
            display: grid;
        }

Grid Template Columns and Rows: Define the number of columns and rows in the grid.

        .container {
            grid-template-columns: repeat(3, 1fr); /* 3 equal-width columns */
            grid-template-rows: 100px 200px; /* 2 rows with different heights */
        }

Grid Gap: Defines the space between grid items.

        .container {
            gap: 10px; /* Adds space between rows and columns */
        }

Placing Items: Items can be placed explicitly by specifying the row and column lines.

        .item {
            grid-column: 1 / 3; /* The item spans from column 1 to column 3 */
            grid-row: 2 / 4;    /* The item spans from row 2 to row 4 */
        }

Other Common Grid Properties:

  • grid-template-areas: Allows you to name grid areas for better readability.
    •         .container {
            grid-template-areas: "header header header"
                                 "sidebar main main"
                                 "footer footer footer";
        }
  • align-items: Aligns grid items along the vertical (column) axis.
  • justify-items: Aligns grid items along the horizontal (row) axis.
CSS Animations

CSS Animations allow you to create smooth transitions from one style to another over a specified duration.

Basic Syntax: CSS animations are defined with two key components: keyframes and the animation property.

        @keyframes animation-name {
            from {
                /* Initial styles */
            }
            to {
                /* Final styles */
            }
        }
        
        element {
            animation: animation-name duration timing-function delay iteration-count direction;
        }

Example: A simple animation that changes the background color from red to blue:

        @keyframes changeColor {
            from {
                background-color: red;
            }
            to {
                background-color: blue;
            }
        }
        
        div {
            animation: changeColor 2s ease-in-out infinite;
        }

Properties used in animations:

  • animation-name: The name of the keyframe animation.
  • animation-duration: Specifies how long the animation lasts.
  • animation-timing-function: Defines the speed curve of the animation (e.g., ease, linear, ease-in, etc.).
  • animation-delay: Sets the delay before the animation starts.
  • animation-iteration-count: Specifies the number of times the animation will play (e.g., infinite for an infinite loop).
  • animation-direction: Defines whether the animation runs forward, backward, or alternates.
CSS Transitions

CSS Transitions allow you to change property values smoothly (over a given duration) when a certain event occurs, like hovering over an element.

Basic Syntax: To create a transition, you need to define the transition property on an element and specify the property, duration, timing function, and delay.

        element {
            transition: property duration timing-function delay;
        }

Example: Changing the background color when the mouse hovers over an element:

        div {
            background-color: red;
            transition: background-color 0.5s ease-in-out;
        }
        
        div:hover {
            background-color: blue;
        }

Properties used in transitions:

  • transition-property: Specifies which property the transition will affect.
  • transition-duration: Defines the duration of the transition.
  • transition-timing-function: Determines the speed curve (e.g., linear, ease-in-out).
  • transition-delay: Specifies the delay before the transition starts.

Note: Transitions only work when a property value changes, so they are commonly used for hover effects or focus states.

CSS Functions

CSS Functions are built-in functions that allow you to perform calculations or other operations within CSS properties.

Common CSS Functions:

  • calc(): Performs calculations to determine property values, such as adding or subtracting lengths, percentages, etc.
    •         div {
            width: calc(100% - 20px);
        }
  • rgb() and rgba(): Defines colors using the RGB color model. rgba() allows you to set an alpha (opacity) value.
    •         div {
            background-color: rgb(255, 0, 0); /* Red */
        }
        div {
            background-color: rgba(255, 0, 0, 0.5); /* Semi-transparent Red */
        }
  • var(): Used to insert the value of a CSS custom property (variable) into other CSS properties.
    •         :root {
            --main-color: blue;
        }
        
        div {
            color: var(--main-color);
        }
  • url(): Specifies the URL of a resource, like an image, for background images or other properties.
    •         div {
            background-image: url('image.jpg');
        }
  • rotate(), scale(), translate(), skew(): Used for transformations on elements. These functions can modify the element's position, size, and orientation.
    •         div {
            transform: rotate(45deg);
        }
CSS Variables

CSS Variables (also known as custom properties) allow you to store reusable values that can be referenced throughout your CSS. They help to make code more maintainable and flexible.

Basic Syntax: You define a CSS variable using the -- prefix and can use it with the var() function.

        :root {
            --primary-color: #3498db;
            --font-size: 16px;
        }
        
        div {
            color: var(--primary-color);
            font-size: var(--font-size);
        }

Key Points:

  • Scope: Variables defined in :root are globally accessible, while those defined within specific elements are scoped to those elements.
    •         .container {
            --secondary-color: #e74c3c;
        }
        div {
            color: var(--secondary-color);
        }
  • Fallback values: You can define fallback values for variables in case they are not defined.
    •         div {
            color: var(--undefined-color, black); /* If --undefined-color is not set, use black */
        }
  • Inheritance: CSS variables inherit values from their parent elements, making them easy to apply consistently throughout the DOM.
